# Обучение нейросетей различной архитектуры для MNIST

Соответствующие примеры необходимо скопировать в директорию `%CNTK%\Examples\Image\GettingStarted` и запускать
оттуда командой
```
cntk configFile=<filename>.cntk
```

При этом предварительно необходимо загрузить данные MNIST и сформировать обучающие файлы в формате CNTK, запустив
файл `install_mnist.py` из директории `%CNTK%\Examples\Image\DataSets\MNIST` (здесь `%CNTK%` - директория,
куда установлен Cognitive Toolkit).
