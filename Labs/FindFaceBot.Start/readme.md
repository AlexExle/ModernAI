# Лабораторная работа: "Happy Gates"

### Цель лабораторной работы
разработать чат-бота, который будет находить в интернет самые эмоционально-яркие
фотографии указанного человека. Для этого мы используем:
 * Bing Image Search API
 * Cognitive Services Emotion API

### Основные шаги
Данный проект содержит в себе шаблон бота, использующего диалоги. Также в нём уже запрограммирована простая функция для
вызова Bing Search для поиска изображений.

Вам необходимо выполнить следующие шаги:

1. Подумать, как вы будете получать входные данные от пользователя. Здесь есть несколько вариантов:
    - использовать стандартные диалоги (PromptDialog.Text и PromptDialog.Choice)
    - использовать FormFlow
    - использовать LUIS и анализ фраз типа *Please, find happy face of Bill Gates*

2. Реализовать функцию вызова Emotion API для 7-10 картинок, полученных как результат поиска. Для этого надо будет 
   подключить nuget-пакет `Microsoft.ProjectOxford.Emotion` и использовать следующий код для работы с эмоциями:

```cs
var EmotionClient = new EmotionServiceClient(Config.EmotionKey);
var r = await EmotionClient.RecognizeAsync(url);
if (r.Length>0)
{
    var happy = r[0].Scores.Happiness;
    ...
}

```

3. Если останется время, попробуйте реализовать хранение контекста разговора и кеширование результатов запросов для конкретных знаменитостей.
