# FaceRecognitionTracker

## Учимся определять эмоции по камере

Начальный вариант приложения находится в директории [FaceRecognitionTracker.Start]. Окончательный вариант,
который у вас должен получится - в [FaceRecognitionTracker.End].

## Анализируем эмоции в облаке

Те из вас, кто интересуются также темой интернета вещей, могут продолжить выполнять эту работу
и построить анализатор эмоций в облаке. Для этого, перейдите в репозиторий
http://github.com/evangelism/FaceRecognitionTracker.
